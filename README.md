# University Canteen Management System

## Project Overview

This project is a web-based University Canteen Management System developed using Flask and MySQL.

It helps administrators manage food items and customer orders efficiently.

---

## Features

- Admin Login
- Add Food Items
- Delete Food Items
- View Food Menu
- Customer Ordering
- Order Management
- Daily Reports
- Responsive User Interface

---

## Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- MySQL
